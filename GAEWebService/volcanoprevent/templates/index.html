API DOC

Volcanoes
-GET- 'http://volcanoprevent.appspot.com/volcanoes'
returns a list of all the volcanoes in the database as json with the next syntax {"volcanoes": [list containing all the volcanoes ( {"country": (str), "size": (int), "location": (lat(str),lon(str)), "name":(str)} )] }

Evacuation Routes

-GET- 'http://volcanoprevent.appspot.com/evaRoute/(latitude(double))_(longitude(double))'
returns the closes Evacuation Route to the latitude and longitude specified in the URL, Syntax:
{"origin":position(str),"route":[list of positions "lat, lon"(str)],"name":(str)}
like so:
{"origin":"28.673898,-106.079871","route":["28.673898,-106.079871","28.673896,-106.080159","28.674125,-106.080264","28.673987,-106.08071"],"name":"Entrada lateral"}

Meeting Points

-GET- 'http://volcanoprevent.appspot.com/evaRoute/name(str)'
will return the MeetingPoint with the name specified with the next syntax:
{"location":"lat,lon"(str),"name":(str)}
like this:
{"location":"28.674295,-106.076957","name":"Gimnasio ITESM"}
All meeting points can be retrieves by passing a * instead like this:
'http://volcanoprevent.appspot.com/evaRoute/*'
the result will have the next syntax:
{"meetingpoints":[ list of all the meeting points {"name":(str),"location":"lat,lon"(str)},]}
like this:
{"meetingpoints":[{"name":"lol","location":"28.674351,-106.075324"},{"name":"Zocalo","location":"0.0,0.0"},{"name":"hola","location":"18.57,-95.32"},{"name":"Gimnasio ITESM","location":"28.674295,-106.076957"}]}
